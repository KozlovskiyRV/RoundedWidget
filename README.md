# RoundedWidget

Кастомный Qt Widgets компонент с поддержкой интеграции в **Qt Designer**.

---

## 📁 Структура проекта

```
RoundedWidget/
│
├── roundedwidget/           # библиотека виджета
│   ├── roundedwidget.h
│   ├── roundedwidget.cpp
│   └── roundedwidget.pro
│
├── roundedwidgetplugin/     # плагин для Qt Designer
│   ├── roundedwidgetplugin.h
│   ├── roundedwidgetplugin.cpp
│   ├── icons.qrc
│   ├── roundedIcon.png
│   └── roundedwidgetplugin.pro
│
└── RoundedWidget.pro        # SUBDIRS-проект
``
